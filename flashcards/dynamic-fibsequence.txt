=== FIBONACHI SEQUENCE ===

FIB(n) = FIB(N-1) + FIB(N-2)

Base Cases:

N <= 2, RETURN 1

*** Resursive Solution ***

const fib = (n) => {

    if(n <= 2) return 1;

    return fib(n - 1) + fib(n - 2);
};

* Time Complexity: O(2^n)
* Space Complexity: O(n)


*** Dynamic Solution using Memoization ***

const fib = (n, memo = {}) => {

    if(n in memo) return memo[n];
    if(n <= 2) return 1;

    memo[n] = fib(n - 1, memo) + fib(n - 2, memo);
    return memo[n];
};
