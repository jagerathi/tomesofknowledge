*** Top Commands ***

    * killall <name> 
    * touch

        * touch file2.txt file3.txt  // creates multiple file 
        * touch <filename> // for an existing file, will change the last modified date on the file 
        * sudo touch /forcefsck // will run a file check on next reboot
